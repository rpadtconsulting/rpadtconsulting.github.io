# Finance Management System

Sistema de Gestão Financeira desenvolvido em HTML/JS puro,
integrado com Power Automate via APIs REST.

## Funcionalidades
- Dashboard com KPIs, gráfico pizza por categoria e análise mensal
- Gestão de Lançamentos (CRUD + anexos + visualização de PDF/imagens)
- Gestão de Clientes (CRUD + busca de CEP + seletor de país)
- Autenticação via Power Automate
- Sidebar responsiva com navegação entre páginas

## Tecnologias
- HTML5 / CSS3 / JavaScript (vanilla)
- Chart.js 4.4
- Font Awesome 6.5
- BrasilAPI (consulta de CEP)
- Power Automate (backend/APIs)
